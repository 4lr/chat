## System requirement

- JDK 8
- Docker

## Available Scripts

In the project directory, you can run:

#### `docker-compose up -d`

Ups the MongoDB

#### `./gradlew bootRun`

Runs the backend app in the development mode.<br>
Open [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html) to view swagger in the browser.
